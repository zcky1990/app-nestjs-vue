## Description

this is project that use nestJS handlebar engine + vueJS
## Installation

```bash
$ npm install
```

## Running the app

running nestJS
```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

Running webpack hotreload
```bash
# development mode
$ npm run build:dev

# production mode
$ npm run build:prod
```

# Running StoryBook
````
npm run storybook
````
